# 抗体库存管理系统（Antibody Inventory Manager）

纯前端工具：用于在浏览器中管理抗体库存与存放位置。

## 使用
- 直接打开 `index.html`
- 数据默认保存在浏览器 LocalStorage
- 支持导出/导入 JSON 备份

## 部署（GitHub Pages）
放入仓库路径：`tools/antibody-inventory/`，并通过 GitHub Pages 访问：
`https://<username>.github.io/<repo>/tools/antibody-inventory/`

## 注意
- 建议使用现代浏览器（Chrome/Edge/Safari/Firefox）
