# Lab Tools

实验室小工具集合（Lab Tools）  
基于 GitHub Pages 发布，数据本地存储，离线优先设计。

访问地址：

https://sonify2026.github.io/lab-tools/

---

## 项目理念

本项目旨在提供一组：

- 轻量级
- 无后端依赖
- 无账号系统
- 数据完全本地保存
- 可长期自托管

的小工具，用于科研与个人效率管理。

所有工具均为纯前端应用，直接在浏览器运行。

---

## 当前工具列表

### 1. 细胞冻存管理系统
路径：`/lab-tools/tools/cell-storage/`

功能：
- 容器 / 孔位结构管理
- 全局搜索
- 低库存预警
- 导入 / 导出备份

数据存储方式：
- localStorage（浏览器本地）

---

### 2. 抗体库存管理系统
路径：`/lab-tools/tools/antibody-inventory/`

功能：
- 抗体位置可视化管理
- 拖拽移动
- 剩余量计算
- 低库存预警
- 导入 / 导出备份

数据存储方式：
- localStorage（浏览器本地）

---

### 3. 离线密码库
路径：`/lab-tools/tools/vault/`

特性：
- IndexedDB 本地存储
- PBKDF2 + AES-GCM 加密
- 自动锁定机制
- 加密导入 / 导出
- 账号绑定信息管理
- 会员到期提醒

安全设计说明：
- 主密码仅用于派生密钥
- 不保存明文密码
- 列表缓存不常驻 secret 字段
- 无任何网络请求

重要提示：
> 若主密码丢失，数据无法恢复。

---

### 4. IC50 计算器

路径：`/lab-tools/tools/ic50-calculator/`

用途：

- 细胞毒性实验
- 药物抑制曲线分析
- 剂量-反应关系建模

功能：

- 四参数 Logistic（4PL）拟合
- 自动计算 IC50
- 对数坐标支持
- 拟合曲线可视化

说明：

- 所有计算均在浏览器本地完成
- 不上传实验数据

### 5. 摩尔计算器

路径：`/lab-tools/tools/molarity_calculator/`

用途：

- 基础化学换算
- 溶液配置计算
- 实验前定量准备

功能：

- 自动解析化学式（支持括号 / 水合物）
- 摩尔质量（M）自动计算
- 质量 m / 物质的量 n 换算


## 数据与隐私说明

- 本项目没有服务器。
- 所有数据仅存储在使用者的浏览器中。
- 不收集、不上传、不分析任何用户数据。
- 不包含统计或追踪代码。

若清除浏览器数据、使用无痕模式或更换设备，数据不会自动同步。

请务必自行备份。
